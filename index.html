Game Description

Mavis Disappearance is a cartoonish themed Hack and Slash role playing game. Experience the thrilling actions with an isometric character point of view, boss battles and repeatable quests each stage as you embark on an an epic adventure where you can play as zeref and take on foes and unlock secrets on finding the lost princess, Mavis

Minimum System Requirements

  • 2 GHz processor (supporting SSE2 instruction set or higher)
  • 1 GB RAM (2 GB of RAM for Windows Vista and newer)
  • 8 GB available hard disk space
  • Shader version 2.0 capable video card
  • Screen resolutions up to 1920x1200
  • Support for DirectX v9.0c or better
  • Windows 7, Windows 8
